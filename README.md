# Zombie Arena (Orb Arena)

A browser-based arcade survival game featuring:
- Progressive waves
- Mini-boss and Boss mechanics
- Dynamic telegraphed attacks
- Powerups (Shield, Multi-shot, Rapid fire, Slow, Heal, Nuke)
- WebAudio-based adaptive soundtrack
- Farcade SDK integration (ready/gameOver/haptics)

## Controls
Desktop:
- WASD to move
- Space or Hold Click to shoot

Mobile:
- Left joystick to move
- Auto Shoot is enabled for mobile
