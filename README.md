# 🧟 Zombie Arena (Orb Arena)

A fast-paced browser-based survival shooter built with pure HTML5 Canvas and WebAudio.

Fight escalating waves of zombie orbs, survive mini-boss and boss rounds, and collect powerups to stay alive.

🎮 **Play Here:**  
https://jay-jay-tee.github.io/zombie-arena-game/

---

## 🎯 Game Features

### ⚔️ Progressive Wave System
- Increasing difficulty scaling
- Special Mini-Boss waves at 5, 15, 25...
- Main Boss waves at 10, 20, 30...
- Dynamic spawn logic

### 👹 Boss Mechanics
- **Mini-Boss**
  - Dash attacks
  - Ground spikes
  - Ring projectiles
  - Zombie spawning

- **Main Boss**
  - Laser sweep
  - Teleport + slam
  - Healing phase
  - Projectile storms
  - Speed surge

### ⚡ Powerups
- 🛡 Shield
- 🔫 Multi-Shot
- 🚀 Rapid Fire
- ❄ Slow Enemies
- ❤️ Heal
- ☢ Nuke

### 🔔 Telegraph System
Boss attacks are visually telegraphed before execution to reward player reaction.

### 🎵 Adaptive Audio Engine
- Custom WebAudio implementation
- Dynamic background loops
- Context-based sound effects
- Boss warning sounds
- Countdown effects

### 📱 Cross-Platform Controls

**Desktop**
- `WASD` → Move
- `Space` or Hold Click → Shoot

**Mobile**
- Left joystick → Move
- Auto-aim shooting
- Touch-based controls

---

## 🧠 Technical Highlights

- Built entirely with vanilla JavaScript
- HTML5 Canvas rendering
- Custom sprite sheet generation
- Real-time particle system
- Delta-time based game loop
- State-machine driven game flow
- Forward attack telegraph rendering
- WebAudio-based soundtrack (no audio files used)

No external game engine. No frameworks.
